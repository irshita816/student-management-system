
# Student Management System

A simple and interactive Student Management System developed using HTML, CSS, and JavaScript. This application allows users to manage student records efficiently through a clean and user-friendly interface.

## Features
- Add new student records
- Search students by name
- Sort students alphabetically
- Delete individual student records
- Delete all records
- Display total number of students
- Store data using Local Storage
- Responsive and easy-to-use interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Learning Outcomes
- DOM Manipulation
- Event Handling
- Arrays and Objects
- CRUD Operations
- Local Storage
- Search and Sorting Algorithms
